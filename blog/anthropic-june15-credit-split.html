What Actually Changes

Anthropic is drawing a line between "interactive" Claude usage (chat, Cowork, terminal) and "programmatic" usage (Agent SDK, GitHub Actions, third-party apps). The programmatic side gets its own credit bucket:

Credits don't roll over. They're billed at API rates. And when they're gone, one of two things happens: either you pay extra at full API rates, or your agents just... stop.

Who's Affected

Anthropic named the third-party tools directly: OpenClaw, Conductor, Zed, Jean. If you use any of these with a Claude subscription, June 15 is when your workflow gets a paywall it didn't have before.

The Rent Pattern, Accelerated:

  1. April: Anthropic bans third-party agents from subscriptions
  2. May: Tests removing Claude Code from Pro
  3. June 15: Restructures pricing — same cost, less access, separate credits

Three changes in three months. Every single one takes something away from subscribers.

The Pattern Nobody's Naming

This is the third time Anthropic has changed the rules for Claude subscribers this year. Each change followed the same playbook:

  1. Let people build on your platform (lock-in)
  2. Restrict who they can build with (control)
  3. Charge them separately for what they already paid for (extraction)

Boris Cherny, head of Claude Code at Anthropic, called third-party tools "really hard to do sustainably." He's right — for Anthropic's margins. But that's exactly the point. When a vendor's sustainability and your access are in conflict, the vendor wins. Every time.
